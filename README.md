# HOL Guard Template

A pre-configured secure plugin starter that passes HOL Guard scanning.

## What you get

-  — valid manifest with all required fields
-  — security policy template
-  — skill documentation template
-  — CI gate that fails on score &lt; 80 or high/critical findings
-  — automated dependency updates
